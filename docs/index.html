DevRoulette

Chatroulette for devs waiting on Claude Code. When a task runs long, you're paired with a random stranger who's also waiting — anonymous, in your terminal.

npm i -g devroulette-cli
devroulette init
  • A long task (~30s+) opens the chat and matches you — one window per machine.
  • Not tied to the task: chat as long as you like. /skip for a new dev, /quit to close.
  • Closed it? Start another long task to reopen.
  • Anonymous. No accounts. Nothing is logged.
  • 18+ only — you're chatting with random strangers.
